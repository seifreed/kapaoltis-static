# Imágenes

Coloca aquí las imágenes de tu sitio web.

## Imágenes necesarias:

### Slider (Hero):
- `slide1.jpg` - Primera imagen del slider (recomendado: 1920x800px)
- `slide2.jpg` - Segunda imagen del slider (recomendado: 1920x800px)
- `slide3.jpg` - Tercera imagen del slider (recomendado: 1920x800px)

### Tarjetas de la página principal:
- `tradicio.jpg` - Imagen para la tarjeta "Tradició" (recomendado: 600x400px)
- `cultura.jpg` - Imagen para la tarjeta "Cultura Popular" (recomendado: 600x400px)
- `comunitat.jpg` - Imagen para la tarjeta "Comunitat" (recomendado: 600x400px)

## Cómo obtener las imágenes de WordPress:

1. **Desde el panel de WordPress:**
   - Ve a Medios > Biblioteca
   - Descarga las imágenes que uses en tu sitio actual

2. **Usando el navegador:**
   - Visita tu sitio WordPress
   - Click derecho en cada imagen > "Guardar imagen como..."
   - Guárdala con el nombre correspondiente

3. **Optimizar imágenes (opcional pero recomendado):**
   - Usa herramientas online como TinyPNG o Squoosh
   - Reduce el tamaño de archivo manteniendo la calidad
   - Esto hará que tu sitio cargue más rápido

## Placeholder temporal:

Si aún no tienes las imágenes, puedes usar servicios de placeholder:
- https://placehold.co/1920x800 (para sliders)
- https://placehold.co/600x400 (para tarjetas)

O genera imágenes con colores sólidos temporalmente hasta que tengas las reales.
